# 🖼️ Web-Based Image Editor

A modern, responsive web-based image editor built with **HTML**, **CSS**, and **JavaScript**—no backend or installation required. The app allows users to edit images directly in the browser with intuitive controls and built-in cybersecurity tools.

## 🌐 Live Demo

🔗 [Click here to try the Image Editor](https://image-modifier.netlify.app/)

## 🚀 Features

### 🎨 Image Editing Tools
- **Upload Images** from your local system
- Apply filters:
  - Brightness
  - Contrast
  - Hue Rotation
  - Saturation
  - Grayscale
  - Blur
- **Transform Images**:
  - Rotate (left/right)
  - Flip (horizontal/vertical)
- **Channel Isolation**: View individual Red, Green, or Blue color channels

### 🔐 Cybersecurity Utilities
- **Scan for Malware**: Simulates a simple check to validate uploaded file integrity
- **Watermarking**:
  - Add visible text watermarks
  - Remove existing watermarks (if applied through the app)

### 🧭 UI & Usability
- Controls are grouped in a **collapsible sidebar** for streamlined access
- **Responsive Design**: Works smoothly on both desktop and mobile devices
- **Instant Preview** of all edits
- **Client-Side Only**: No server or external dependencies
